Algorithm:
-set nilai dari siswa dengan 0
-lakukan loop selama siswa kurang dari 40
    -jika kuku siswa panjang
        -lakukan hukuman
        -increment siswa
    -jika kuku siswa tidak panjang
        -lakukan pujian
        -increment siswa

Pseudocode:
SET "siswa" value with 0
WHILE "siswa" < 40
    IF "siswa" kuku == panjang
        DO "hukum"
        ADD "siswa" value by 1
    ELSE
        DO "puji siswa"
        ADD "siswa" value by 1
    END IF
END WHILE
